# Etapa 03 - Materiais de Apoio e Slides

Esta etapa é reservada para transformar o roteiro em material de apresentação e apoio.

Exemplos do que deve entrar aqui:
- slides da palestra;
- material de apoio para participantes;
- exemplos visuais e comparações de prompts;
- versão resumida ou handout, se necessário.

## Dependência principal

Esta etapa depende da consolidação de [Etapa 02 - Roteiro e Conteúdo](../02-roteiro-e-conteudo/README.md).
