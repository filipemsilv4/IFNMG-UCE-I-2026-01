# Etapas do Projeto

Esta pasta concentra o trabalho organizado por etapa, com nomes explícitos e alinhados ao fluxo real do projeto.

## Etapas

- [01-levantamento-de-referencias](01-levantamento-de-referencias/README.md): primeira atividade de Paulo, focada em pesquisa, curadoria e base conceitual.
- [02-roteiro-e-conteudo](02-roteiro-e-conteudo/README.md): transformação da pesquisa em narrativa, roteiro e estrutura da palestra.
- [03-materiais-de-apoio-e-slides](03-materiais-de-apoio-e-slides/README.md): produção de slides, exemplos visuais e material complementar.
- [04-divulgacao-e-inscricoes](04-divulgacao-e-inscricoes/README.md): estratégia de divulgação, inscrições e comunicação com participantes.
- [05-execucao-da-palestra](05-execucao-da-palestra/README.md): preparação técnica, apresentação e coleta de presença e feedback.
- [06-encerramento-e-relatorio](06-encerramento-e-relatorio/README.md): análise final, certificados, relatório e lições aprendidas.

## Como usar

- Documentos de referência geral continuam em `01-contexto` e `02-planejamento`.
- Trabalho operacional e intelectual do projeto deve preferencialmente nascer dentro da etapa correspondente.
- Quando uma etapa depender de outra, o vínculo deve ser registrado no README da própria etapa.
