# UCE no curso de Computação

Resumo derivado do material `UCE - Computação.pdf`.

## Síntese institucional

- A extensão no IFNMG é tratada como processo educativo articulado ao ensino e à pesquisa.
- No curso de Ciência da Computação, as atividades de extensão são ofertadas por meio das UCEs.
- As UCEs representam `11,25%` da carga horária total do curso.
- As ações devem seguir diretrizes como:
  - indissociabilidade entre ensino, extensão e pesquisa;
  - interdisciplinaridade e interprofissionalidade;
  - interação dialógica;
  - impacto na formação discente;
  - impacto na transformação social.

## Origem das demandas

As demandas das UCEs podem surgir de:
- comunidade externa ao campus;
- instituições públicas de ensino municipais e estaduais;
- setor produtivo local;
- Portfólio de Oportunidades do IFNMG.

## Implicações práticas para este projeto

- A palestra precisa ter conexão real com uma demanda externa e socialmente justificável.
- O projeto não deve ser apenas expositivo; ele precisa demonstrar valor formativo e impacto para o público atendido.
- Os registros finais podem incluir produtos e evidências como:
  - relatórios;
  - portfólios;
  - vídeos;
  - artigos;
  - exposições com textos e imagens.

## Relevância para o repositório

Este repo foi organizado para facilitar exatamente esse tipo de evidência:
- planejamento;
- conteúdo produzido;
- referências usadas;
- decisões registradas;
- base para relatório final da UCE.
